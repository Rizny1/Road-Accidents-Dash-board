# Road Accident Dashboard (Excel)

This project is an interactive Road Accident Dashboard developed using Microsoft Excel.  
It provides visual insights into road accident trends, severity levels, locations, and time-based patterns to support data-driven analysis and decision-making.

## Features
- Interactive charts and KPIs
- Accident trend analysis
- Severity and location-based insights
- Time-based comparisons
- User-friendly Excel dashboard

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts and Slicers

## File Information
- Dashboard file format: `.xlsb`
- Optimized for reduced file size and performance

## How to Use
1. Download the Excel dashboard file
2. Open using Microsoft Excel
3. Use slicers and filters to explore the data
